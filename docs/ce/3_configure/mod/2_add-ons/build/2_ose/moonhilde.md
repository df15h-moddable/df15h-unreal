# Moon Wildlife

by Moonhilde.

Q: Why?

A: Moonhilde is a "modding legend" for Conan Exiles, for this mod and for the "Sapphire Exile" map.

Q: Is it up-to-date?

A: NO, not since Jun 2023, but my own experience (in 2025) is still positive.
