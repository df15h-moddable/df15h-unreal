# Accessory Wardrobe

by Whiskey

[Accessory Wardrobe](https://steamcommunity.com/sharedfiles/filedetails/?id=2098700751) @ Steam Workshop.

Features ...

- Works for the player character, thralls, and Pippi thespians.
